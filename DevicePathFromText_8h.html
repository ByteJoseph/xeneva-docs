Detailed Description

EFI_DEVICE_PATH_FROM_TEXT_PROTOCOL as defined in UEFI 2.0. This protocol provides service to convert text to device paths and device nodes.

Convert text to the binary representation of a device node.

Parameters
TextDeviceNodeTextDeviceNode points to the text representation of a device node. Conversion starts with the first character and continues until the first non-device node character.
Return values
a_pointerPointer to the EFI device node.
NULLif TextDeviceNode is NULL or there was insufficient memory.
