Detailed Description

The protocol provides information about the version of the EFI specification that a driver is following. This protocol is required for EFI drivers that are on PCI and other plug-in cards.

typedef struct _EFI_DRIVER_SUPPORTED_EFI_VERSION_PROTOCOL EFI_DRIVER_SUPPORTED_EFI_VERSION_PROTOCOL

The EFI_DRIVER_SUPPORTED_EFI_VERSION_PROTOCOL provides a mechanism for an EFI driver to publish the version of the EFI specification it conforms to. This protocol must be placed on the driver's image handle when the driver's entry point is called.
