UINT32 EFI_FIRMWARE_VOLUME_EXT_ENTRY_OEM_TYPE::TypeMask

A bit mask, one bit for each file type between 0xC0 (bit 0) and 0xDF (bit 31). If a bit is '1', then the GUID entry exists in Types. If a bit is '0' then no GUID entry exists in Types.
