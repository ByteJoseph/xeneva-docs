◆ EFI_DRIVER_FAMILY_OVERRIDE_GET_VERSION

typedef UINT32(EFIAPI * EFI_DRIVER_FAMILY_OVERRIDE_GET_VERSION) (IN EFI_DRIVER_FAMILY_OVERRIDE_PROTOCOL *This)

This function returns the version value associated with the driver specified by This.

Retrieves the version of the driver that is used by the EFI Boot Service ConnectController() to sort the set of Driver Binding Protocols in order from highest priority to lowest priority. For drivers that support the Driver Family Override Protocol, those drivers are sorted so that the drivers with higher values returned by GetVersion() are higher priority than drivers that return lower values from GetVersion().

Parameters
ThisA pointer to the EFI_DRIVER_FAMILY_OVERRIDE_PROTOCOL instance.
Returns
The version value associated with the driver specified by This.
