EFI_STATUS EFI_80211_DISASSOCIATE_DATA_TOKEN::Status

Will be set to one of the following values: EFI_SUCCESS: Disassociation operation completed successfully. EFI_DEVICE_ERROR: An unexpected network or system error occurred. EFI_ACCESS_DENIED: The disassociation operation is not completed due to some underlying hardware or software state. EFI_NOT_READY: The disassociation operation is started but not yet completed.
