Detailed Description

Real Time clock Architectural Protocol as defined in PI Specification VOLUME 2 DXE

This code abstracts time and data functions. Used to provide Time and date related EFI runtime services.

The GetTime (), SetTime (), GetWakeupTime (), and SetWakeupTime () UEFI 2.0 services are added to the EFI system table and the EFI_REAL_TIME_CLOCK_ARCH_PROTOCOL_GUID protocol is registered with a NULL pointer.

No CRC of the EFI system table is required, since that is done in the DXE core.
