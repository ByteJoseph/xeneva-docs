RETURN_STATUS EFIAPI PublishSystemMemory ( IN PHYSICAL_ADDRESS  MemoryBegin,
IN UINT64  MemoryLength 
)

Declares the presence of permanent system memory in the platform.

Declares that the system memory buffer specified by MemoryBegin and MemoryLength as permanent memory that may be used for general purpose use by software. The amount of memory available to software may be less than MemoryLength if published memory has alignment restrictions. If MemoryLength is 0, then ASSERT(). If MemoryLength is greater than (MAX_ADDRESS - MemoryBegin + 1), then ASSERT().

Parameters
MemoryBeginThe start address of the memory being declared.
MemoryLengthThe number of bytes of memory being declared.
Return values
RETURN_SUCCESSThe memory buffer was published.
RETURN_OUT_OF_RESOURCESThere are not enough resources to publish the memory buffer
