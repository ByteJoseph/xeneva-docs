typedef EFI_STATUS(EFIAPI * EFI_PEI_STALL) (IN CONST EFI_PEI_SERVICES **PeiServices, IN CONST EFI_PEI_STALL_PPI *This, IN UINTN Microseconds)

The Stall() function provides a blocking stall for at least the number of microseconds stipulated in the final argument of the API.

Parameters
PeiServicesAn indirect pointer to the PEI Services Table published by the PEI Foundation.
ThisPointer to the local data for the interface.
MicrosecondsNumber of microseconds for which to stall.
Return values
EFI_SUCCESSThe service provided at least the required delay.
